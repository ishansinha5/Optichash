LIVE MATCH ENGINE

This project began as a challenge to build a computer vision pipeline optimized for standard hardware constraints. The core focus is efficient system routing. When you upload a photo below, the architecture deliberately avoids spinning up heavy neural networks right away.

Instead, the payload is routed to a native C++ microservice that calculates a byte-level Perceptual Hash (pHash). If it finds an exact spatial match in the local cache, it returns the metadata instantly—bypassing the machine learning layer entirely. The quantized PyTorch engine is only invoked for truly unrecognized images.
